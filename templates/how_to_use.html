How to Use the Listing App

Getting Started

The Listing App helps you format product titles and descriptions for different sales channels (eBay, Amazon, and Magento). Here's how to get started:

  1. Prepare your CSV file with your product listings
  2. Make sure your CSV has either an 'ItemTitle' or 'Item_Title' column
  3. Optional: Include 'Material', 'Wheelbase', and 'Cab_Type' columns for additional information

CSV File Format

Important: All columns listed below are REQUIRED in your CSV file.

Base Columns:

  • 'ItemTitle' or 'Item_Title' - Your product title
  • 'Material' - Product material information
  • 'Wheelbase' - Vehicle wheelbase information
  • 'Cab_Type' - Vehicle cab type information

Extended Property Columns:

  • 'ALTERATION_Attribute' - Alteration information
  • 'Bar length_Attribute' - Bar length information
  • 'Brackets_Attribute' - Brackets information
  • 'BTS-NOTSOLD_Attribute' - BTS-NOTSOLD status
  • 'Bullet1_Attribute' - First bullet point
  • 'Bullet2_Attribute' - Second bullet point
  • 'Bullet3_Attribute' - Third bullet point
  • 'Bullet4_Attribute' - Fourth bullet point
  • 'Bullet5_Attribute' - Fifth bullet point
  • 'Diameter of bar_Attribute' - Bar diameter information
  • 'eBay ID_Attribute' - eBay identifier
  • 'eBay Store ID_Attribute' - eBay store identifier
  • 'eBay1 Store ID_Attribute' - eBay1 store identifier
  • 'Filter Attribute_Attribute' - Filter information
  • 'Finish_Attribute' - Finish information
  • 'Fitment Type_Attribute' - Fitment type
  • 'Fitting Link_Attribute' - Fitting link URL
  • 'Fitting link video_Attribute' - Fitting video link
  • 'google_product_category_Attribute' - Google product category
  • 'HS Code_Attribute' - HS code
  • 'Included in sale_Attribute' - Items included in sale
  • 'Instructions_Attribute' - Product instructions
Important Notes:
  • All columns must be present in your CSV file
  • Column names are case-sensitive
  • Extended property columns must end with "_Attribute"
  • If you don't have a value for a column, leave it empty but the column header must still exist
CSV Example:
ItemTitle,Material,Wheelbase,Cab_Type,ALTERATION_Attribute,Bar length_Attribute,...
"Grill Bar",Steel,Short,Single,None,1200mm,...
To get these columns, go to Linnworks >> My Inventory >> search for items to update >> click export and choose export type as extended properties >> add ItemTitle (Title from linnworks) to this exported csv and double check to make sure you have all columns >> csv ready to upload

Using the App

  1. Upload Your File

    • Click "Choose File" and select your CSV file
    • Only CSV files are accepted

  2. Select Update Type

    • "Titles Only" - Only process product titles
    • "Descriptions Only" - Only process product descriptions
    • "Titles + Descriptions" - Process both titles and descriptions

  3. Choose Sales Channels

    • eBay 1 - Primary eBay format
    • eBay 0 - Secondary eBay format
    • Magento - Magento store format
    • Amazon - Amazon marketplace format

  4. Select Vehicle Type

    • Choose the appropriate vehicle type (4x4, Truck, Van)
    • This affects which keywords are added to your titles

  5. Fill Additional Information

    • Make - Vehicle make (e.g., Ford, Toyota)
    • Model - Vehicle model (e.g., Ranger, Hilux)
    • Year - Vehicle year or year range
    • Description - Additional description text

Title Formatting

Channel-Specific Formats:

  • eBay 1 & 0: 78-80 characters, includes all keywords
  • Magento: Up to 200 characters, includes only material keywords
  • Amazon: 80-200 characters, includes "Bragan" and "XSKU"
Note: Each channel has specific formatting rules and character limits that are automatically applied.

Keywords and Materials

The app uses two special files to manage keywords and materials:

  • keywords.txt - Contains vehicle-specific and general keywords
  • materials.txt - Maps product types to their materials
Tip: Material keywords are automatically added based on the product type, you don't need to specify them manually.
